# Погодное приложение (Open-Meteo версия)

Приложение для отображения текущей погоды и прогноза на 5 дней для любого города. Использует бесплатные API без ключей.

## Особенности этой версии

- **Без API ключей** - использует Open-Meteo и OpenStreetMap
- **Работает полностью в браузере** - не требует серверной части
- **Геолокация** - автоматическое определение местоположения
- **Быстрый выбор** - популярные российские города
- **Демо-режим** - работает даже при ошибках сети

## Используемые технологии

1. **Open-Meteo API** - данные о погоде
2. **OpenStreetMap Nominatim** - геокодирование (преобразование названия города в координаты)
3. **LocalStorage** - сохранение последнего города
4. **Geolocation API** - определение местоположения пользователя


## Ограничения

- Open-Meteo предоставляет данные только по координатам
- Геокодирование через Nominatim имеет лимиты (1 запрос в секунду)
- При частых запросах могут быть временные блокировки
